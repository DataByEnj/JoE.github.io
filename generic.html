Student/Advisor
Tracking
(Excel)

This Excel project, designed for use by a college department, includes two dashboard tabs that track key metrics such as top and bottom GPAs, frequently applied colleges, top colleges applied to, and the average GPA of academic plans.

COVID-19 Layoffs Analysis (SQL)

This project focuses on analyzing raw layoff data in Microsoft SQL. Its goal is to understand and explore the impact of workforce reductions across various tech sectors from the onset of COVID-19 to the present.

The analysis provides insights into trends, key drivers, and sector-specific challenges, helping viewers understand the pandemic's real-world implications for employment.

SQL Skills Used: Data Filtering, Sorting with ORDER BY, Aggregations, Date Functions, String Manipulation, Common Table Expressions, Window Functions, Ranking Functions (DENSE_RANK), Pattern Matching (LIKE), Alias Usage (AS), Date Segmentation by Year/Month
